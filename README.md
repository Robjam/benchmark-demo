### benchmark-demo

A small demo repo of different benchmarking tools (deno 2, mitata, vitest) to accompany an advent code blog entry.

### setup

you will need:
pnpm
node >= v22
deno  >= v2

start by installing the npm dependencies:
```zsh
pnpm install
```

then run a command (vitest):
```zsh
pnpm run benchmark:vitest
```
